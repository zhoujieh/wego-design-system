## 阴影

### 使用原则

- 如无必要不使用
- 即使使用也从轻（`shadow-xs` / `shadow_04`）开始
- 阴影柔化边缘会削减利落质感

### 判断标准

仅在以下目的使用：

1. 物体重叠需区分层级（优先尝试调整颜色/描边，效果不佳再用阴影）
2. 物体可点击，需增强操作观感

### Token 对照

| 级别 | Token | 用途 |
|------|-------|------|
| 轻阴影 | `--shadow-xs` | 优先选用 |
| 中阴影 | `--shadow-sm` | xs 不满足时 |
| 重阴影 | `--shadow-md` / `--shadow-lg` | popmenu 等需强调层级的场景 |

## 动效

### 使用原则

| 原则 | 说明 |
|------|------|
| 清晰 | 遵循出现/消失/变化的逻辑路线 |
| 高效 | 轻微、快速，不占用额外注意力 |

动效的唯一目的是交代实体出现、消失、变化的路线，若不为这个目的就不应使用。

### 时长档位

| Token | 时长 | 适用 |
|-------|------|------|
| `--duration-instant` | 0ms | 即时反馈 |
| `--duration-fast` | 150ms | 轻微状态切换 |
| `--duration-normal` | 250ms | 标准过渡 |
| `--duration-slow` | 350ms | 面板进入/退出 |
| `--duration-xslow` | 500ms | 复杂动画 |

### 常见动效模板

| 模板 | 描述 |
|------|------|
| 加载 | 内容区域的骨架屏或指示器 |
| 面板进入/退出 | 从底部滑入/滑出 |
| 气泡弹出 | 从触发元素边缘移动 10px + 不透明度 0%->100% |
| 气泡消失 | 原地消失，不透明度 100%->0% |
