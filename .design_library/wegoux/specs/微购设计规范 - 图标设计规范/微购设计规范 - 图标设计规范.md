# 微购设计规范 - 图标设计规范

## 图标分类

| 类型 | 定义 | 特点 |
|------|------|------|
| 基础图标 | 不可再拆分的单一动作/名词图标 | 场景广泛，如删除、搜索 |
| 复合图标 | 由 2+ 个基础元素组合而成 | 场景专一，如加购物车 |

## 命名原则

1. 有固有叫法优先使用（搜索、删除、筛选、设置）
2. 无固有叫法则按功能用法命名（隐藏、显示、私密、公开）

## 绘制参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 标准画布 | 24x24px | 所有图标统一画布 |
| 安全区域 | 20x20px | 图标内容绘制范围 |
| 默认描边 | 1.5px | 常规图标粗细 |
| 加粗描边 | 2.25px | 用于加、勾、叉、箭头 |
| 端点样式 | round | 圆角端点 |
| 圆角规则 | 按角度分级 | 90~120: 1px, >120: 1.5px, <45: 0.5px |

## 复合图标规则

- 主元素 100% 尺寸，居中放置
- 次要元素 <=50% 尺寸
- 次要元素优先放主元素内部或画布右下角

## 使用原则

- **清晰优先**：若图标不能减轻认知负担，就不应使用
- **一致优先**：同类操作使用相同基础图标
- 按钮中仅允许使用特别直观的图标（加、勾、叉、箭头等）

## 尺寸规格

### 线性图标

| 尺寸 | 使用场景 |
|------|----------|
| 24x24 | 导航栏图标、独立功能入口 |
| 20x20 | 搭配 16pt 文字，左右布局 |
| 16x16 | 表单跳转箭头（固定粗细 1.5px） |

### 面性图标

| 尺寸 | 使用场景 |
|------|----------|
| 40x40 | 分类图标、服务头像，用于列表 |
| 28x28 | 应用入口，搭配 12pt 文字上下布局 |
| 24x24 | 底部 Tab 选中态 |
| 12x12 | 展开/收起箭头 |
