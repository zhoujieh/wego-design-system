# UsageHint 尺寸误用复盘

> 角色：复盘材料。读取条件：需要追溯 usageHint 归因时；正式规则只以组件契约和当前设计计划为准。

## 现象

带文字和无文字的宿主场景混用了尺寸修饰类，导致同类控件在同一页面出现不一致尺寸。

## 归因方法

1. 从实际组件契约的 `structurePatterns`、`usageHints` 和变体维度定位最早歧义。
2. 对照同类组件的当前契约，区分共性规则与单点偏差。
3. 只在组件契约修正正式规则；设计计划和运行时仅同步该决定。

## 正式来源

- 组件规则：`.codex/skills/wego-design/components/{slug}.json`。
- 当前任务的消费决定：`_spec/design_plan.json`。
- 当前任务实现：`scene.js`。

本材料不新增组件规则、页面规则或验收规则。
