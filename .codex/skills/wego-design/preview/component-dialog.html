对话框 Dialog

居中模态确认弹窗：text（标题+正文+按钮）、status（状态图标+标题+正文+按钮）、title（仅标题+按钮）、input（标题+输入框+按钮）四种变体。阻断式常驻，用户必须操作后才消失。蒙层默认 30%。input 变体必须使用已注册的 input 组件，勾选项必须使用已注册的 checkbox 组件。

交互演示

点击按钮触发真实浮层 dialog（同屏互斥，新 dialog 替换旧 dialog）。dialog 阻断式常驻，默认只能通过按钮关闭，不会自动消失。

点击按钮关闭；点击遮罩默认不关闭；点击链接默认不关闭；input 变体自动聚焦，回车触发主按钮。
