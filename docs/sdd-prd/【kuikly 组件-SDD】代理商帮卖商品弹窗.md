# 【kuikly 组件\-SDD】代理商帮卖商品弹窗

# 1\. 这个功能要解决什么问题？

供应商设置帮卖方式为“自由加价”时，代理商在帮卖商品时，需要自己设置代理商品的加价金额或比例，以明确自己的帮卖收益，并为系统计算代理售价和佣金提供依据。

# 2\. 用户怎么用？

场景 1：代理商从供应商店铺挑选帮卖商品，点击【立即帮卖】，填写“我的售价”，并查看对应的佣金金额。系统会默认按供应商设置的可加价金额或比例自动计算“我的售价”，代理商也可以快速选择加价金额或比例，还可以自由填写加价金额或比例。

> 代理商小王进入供应商老刘的店铺，发现一款很适合自己粉丝的运动鞋，供应商的售价是 100 元，允许代理商自由加价，想帮卖到自己店铺。于是他点击了【立即帮卖】按钮，默认按供应商设置的加价金额加 100 元，即售价 200 元，每出一单赚 100 元佣金，他觉得佣金太高了，于是把售价改为 150 元，即佣金 50 元。
> 
> 



场景 2：代理商帮卖一段时间后，发现有些商品的售价过高，想要调整一下价格，于是在自己的店铺找到对应的帮卖商品，点击【编辑】，修改售价，并查看对应的佣金金额。

> 代理商小王帮卖的运动鞋售价 150 元，佣金 50 元，但是很多粉丝觉得价格太高了，不愿意下单，于是他决定降价销售，于是找到这个商品，点击【编辑】，把售价改为 119 元，即佣金 19 元。

# 做完怎么算合格？

### AC\-01：默认规则

1. 按供应商设置的“代理可加价金额/比例”自动计算售价，并显示对应的佣金。

2. 输入框下方显示默认加价金额/比例，以及 \+30%、\+20%快捷选项，和“手动输入”文字链

3. 点击“手动输入”弹出“加价设置键盘”，自由填写加价金额或比例。

### AC\-02：价格设置

1. 代理商的售价在供应商供货价的基础上加价。

2. 加价额度和比例默认取供应商的设置，可以修改。

3. 佣金比例最高 300%。

### AC\-03：不同 SKU 价格相同

1. 供应商商品包含多个 SKU，且不同 SKU 的供货价相同时，只显示一个价格。

2. 加价卖弹窗中，直接根据供货价和供应商设置的加价金额/比例计算我的售价和佣金，可直接修改“我的售价”，或通过快捷选项、手动输入修改售价。

### AC\-04：不同 SKU 价格不同

1. 供应商商品包含多个 SKU ，且不同 SKU的供货价不同时，页面显示价格区间。

2. 加价卖弹窗中，直接根据供货价和供应商设置的加价金额/比例计算我的售价（区间）和佣金（区间），不能直接修改“我的售价”，只能通过快捷选项、手动输入修改售价。

### AC\-05：比例校验

1. 加价比例最高 300%，最低 1%，超出时 toast 提示：佣金比例不能小于 1%、佣金比例不能大于 300%。

2. 加价比例支持 2 位小数，已输入 2 位小数后，不能再继续输入第 3 位小数。

### AC\-08：代理商修改帮卖商品

1. 不管商品的帮卖方式是自由定价还是固定佣金，都可以修改帮卖商品。

    1. 如果是固定佣金：标题为“赚佣金”，页面显示我的售价和佣金金额；

    2. 如果是自由定价，标题为“加价卖”，页面显示内容与初次帮卖时一致，但是数据为上一次保存的值。

2. 底部按钮为保存和分享，点击分享拉起分享商品组件。

### AC\-06：保存结果

- 合法配置点击确定后，保存成功并返回商品编辑页。

- 保存帮卖方式、供货价、加价比例和加价金额。

- 再次编辑时，能够查看已设置的售价和佣金。

### AC\-07：供应商修改佣金规则

1. 供应商修改可加价金额/比例：代理商加价金额不变

2. 供应商修改供货价：代理商供货价自动变成最新的，加价金额不变

3. 如商品已经不能帮卖，将自动删除帮卖商品，如已经进入赚佣金页面，保存时提示：该商品不支持帮卖，包含以下情况：

    1. 供应商隐藏供货价

    2. 供应商删除商品价格

    3. 供应商删除商品

    4. 供应商下架商品
