# Wego App 核心场景测试集

本目录用于验证微购设计规则修改后的实际生成效果。

## 使用方式

1. 每次只选择一个测试文件。
2. 新建无历史上下文的会话。
3. 使用最新版技能执行文件中的提示词。
4. 每轮测试前恢复干净产物环境。
5. 同一提示词至少重复执行 3 次，检查结果是否稳定。
6. 测试期间不要修改提示词；需要新增条件时创建新的测试用例。

## 核心场景

1. `001-快捷发布产品.md`
2. `002-动态商品流.md`
3. `003-商家工作台.md`
4. `004-库存与补货.md`
5. `005-价格权限管理.md`
6. `006-系统与交易设置.md`
7. `007-好友管理.md`

这些用例来自当前 `wego-app` 已存在的业务场景，用于覆盖表单录入、内容流、工作台、数据管理、复杂流程、权限配置、设置列表和关系管理等核心页面能力。
